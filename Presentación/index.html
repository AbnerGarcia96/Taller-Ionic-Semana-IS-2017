Descripción del taller

El objetivo de este taller es aprender conceptos básicos sobre el desarrollo de aplicaciones híbridas utilizando Ionic 2, así como la conexión a Google Firebase para manejar nuestra base de datos y autenticación de usuarios

Temario

¿Aplicaciones híbridas?

Evolución

Ionic 2

Es la segunda y más reciente versión de Ionic Framework, el cual sirve para desarrollar aplicaciones móviles a partir de código Front-End

Utiliza HTML5 y CSS para la creación de las vistas y TypeScript para darle funcionalidad a estas vistas

Dependencias

NodeJS y NPM:

NodeJS nos permite utilizar JavaScript del lado del servidor, utilizaremos el gestor de paquetes NPM para instalar nuestras dependencias

$ npm install -g [package]

Dependencias

Apache Cordova:

Traduce el código en JavaScript a código nativo de cada sistema (Java para Android y Obj-C/Swift para iOS)

$ npm install -g ionic cordova

Dependencias

Angular 2:

Framework para JavaScript que nos permite utilizar ES6 para variables con tipos dentro de JavaScript

Dependencias

TypeScript:

Transpilador que pasa el código de ES6 a código JS

$ npm install -g typecript typings

Dependencias

Google Firebase:

Servicio que ofrece Google para el manejo de una base noSQL, autenticación de usuarios, manejo de multimedia, etc.

$ npm install angularfire2 firebase --save

Iniciar un nuevo proyecto en Ionic:

$ ionic start --v2 [nombreProyecto] [template]
